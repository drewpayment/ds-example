# Avatars

Simple component that shows a profile photo if one is uploaded, or a color circle with the employee initials inside.

Input | Type | Description | Default value ----------------- | -------------------- |--------------------------------------------------------------------------------------------------- | -------------- `size` | `number` _Required_ | Starts at 24 (pixels), and iterates by 12 | none `firstName` | `string` _Optional_ | first initial reference | none `lastName` | `string` _Optional_ | last initial reference | `name` | `string` _Optional_ | Allows a full name to ba passed in. It must be in Last name, First name format to get the initials | `color` | `string` _Optional_ | Color avatar should take | Blue `#0A7595` `resource` | `string` _Optional_ | Path to image avatar | `vendor` | `boolean` _Optional_ | If vendor is added as true, the avatar will show the business icon rather than an avatar. |
