# Confirm Dialog

Confirm modals serve as a stop gap for accidentally deleting a record.

## Use The confirm modal asks if the user is sure they want to delete the record. The primary action is cancel, and located in the lower right corner. There is no cancel icon in the upper corner. The user needs to choose one option, Remove or Cancel. _Sample Text:_ Are you sure you want to delete this task?
Values | | | ----------| ------------------- | ----------------------------------------------------------------------------------------------------------- | `title` | Optional | Top section of the message. Usually the "Are you sure..." section. This communicates the action being done. | `message` | Optional | Detail text shown muted and italicised | `confirm` | Default of "Delete" | What the confirm button text should say. Usually "Delete" or "Remove" | ## Communication Dialog
## Ds Component _Can be used with NGX or AJS_
Confirm Modal will automatically build the modal based on attributes assigned to the delete button. It needs to be imported into the component file. ## Legacy Support There is currently no legacy support. Use a javascript alert on any page where Angular is not available.
