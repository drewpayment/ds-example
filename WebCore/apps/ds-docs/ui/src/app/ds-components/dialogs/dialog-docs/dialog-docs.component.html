# Dialogs

Dialogs are also called modals, depending on the enviornment.

## Sizes Below are max-width values and use cases for modals. The modal will never exceed the media width. **Small** Width: 300px
*Example: DsConfirm messages* **Medium** Width: 500px
*Example: Simple form modals with a few fields* **Large** Width: 800px
*Example: Complex form modals with two or more columns* ## Methodology When opening a dialog with form elements, put focus on the first element. The positive action button needs to be first in the markup in the actions area. ## Layout Dialogs use an `h2` title. Show `Delete` if there is data to delete. Otherwise, show `Cancel`.
