# Buttons ## Terminology and Placement Placed in the last card on the page. In some cases, the buttons will go in a seperate card, like the employee section. ### Positive Position in the lower right corner. Primary color denotes a primary action on a page. Outline-primary denotes a secondary action on a page. - **Save** Saving to the database. User can immediately see and edit results. - **Continue** Moving to another page of the same form or workflow. No database call happens. - **Submit** User enters data into system, usually into a function or something less tangible than a form. User will not be able to easily see results. - **Create** Used when running a report.
### Negative Position in the lower left corner. While the classes look the same, using the approriate class will give flexibility to make a future change. - **Cancel** Used to cancel a pending action. - **Delete** Removing data from database.
