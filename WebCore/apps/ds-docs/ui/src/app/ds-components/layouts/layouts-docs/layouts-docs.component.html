# Layouts
## Alignment Easily align fields that don't have a label but need to be inline. Use `inline-*-control` where * is the same as the breakpoint used in the row layout. In this example, `md` is used so the class would be `inline-md-control`.
## Add scroll to column based on other columns's height The card that has `.match-height` declared will take the height of the other column in the row. Scroll content needs to be wrapped in `.overflow-y-scroll-container`. `.match-height` supports breakpoint sizes. If column breakpoints are being used, that breakpoint needs to be added here.
## Text Break Use when you want to selectively break text when the parent element has less width.
